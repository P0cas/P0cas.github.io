Modern-day cloud storage issues

The problem with all cloud storage currently provided by Naver, Google, Amazon, Microsoft, etc. is that the company that provides the service has the right to access personal/company information. An individual’s diary (private life), a company’s work list (business confidential contents), etc. are stored in the cloud service. But why does the company that provides it have the right to access the user’s file? I don’t know. Employees can access information to individuals/company if they wish. What if the server that provides cloud services gets hacked? Hackers can steal access to files. However, if the cloud storage is decentralized, there is no problem.
